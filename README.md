# Heart Disease Prediction System

## Project Description
This project uses machine learning algorithms to predict the likelihood of heart disease based on patient health parameters such as age, cholesterol levels, blood pressure, and other medical features.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn

## Machine Learning Algorithms
- Logistic Regression
- K-Nearest Neighbors
- Decision Tree
- Random Forest
- Support Vector Machine (SVM)

## Project Workflow
1. Data loading and preprocessing
2. Data visualization and exploratory data analysis
3. Feature scaling using StandardScaler
4. Training multiple machine learning models
5. Model evaluation using accuracy and classification metrics

## Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Multiple model comparison
- Performance evaluation
